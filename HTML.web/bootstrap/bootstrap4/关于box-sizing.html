使用 box-sizing 样式属性

注意：boostrap4.0盒模型为了在 CSS 中更直观的设置尺寸，将全局的 box-sizing 值从 content-box 修改为 border-box。 这就确保了 padding 不会影响元素最终的宽度计算, 就是说最终大小是包括最外层的 border 的， 如果增加padding，盒的大小不变，只会挤压内容的空间 如果这样设置 box-sizing: content-box; 说明内容空间的大小不变，修改padding会使盒子大小变化。 还有一个值：box-sizing: inherit; 规定应从父元素继承 box-sizing 属性的值。
