Java 语言是一种特殊的高级语言，它即有解释型语言的特征，也具有编译型语言的特征
因为java程序要经过先编译，后解释两个步骤

一般来说，计算机高级语言按程序的执行方式可以分为编译型和解释型两种
如C，C++等高级语言都属于编译型语言
如php，python, ruby 等语言都属于解释型语言

java程序需要经过先编译后解释两个步骤
但这个编译步骤并不会直接生成特定平台的机器码，而是生成一种与平台无关的字节码（也就是*.class文件）
这种字节码必须使用java解释器来解释执行（就是java虚拟机即JVM）

整个过程是这样的：
java源文件（*.java）-->使用 javac 编译 --> 生成 *.class 文件 --> 使用 java 解释执行

只要为不同平台实现了相应的虚拟机，编译后的java字节码就可以跨平台运行
