当一个算术表达式中包含多个基本类型的值时，整个算术表达式的数据类型将发生自动提升。
所有的byte类型、short类型和char类型将被提升到 int 类型。
整个算术表达式的数据类型自动提升到与表达式中最高等级操作数同样的类型。
操作数的等级排列如下所示：
            char--> 
	        int-->long->float->double
byte-->short-->
下面的程序将发生错误
short Sv = 5;
Sv = Sv - 2; 这个表达式中的右边类型提升为 int 类型，将一个int类型值赋给 short类型变量行不通
表达式的类型将严格保持和表达式中最高等级操作数相同的类型。
即使无法除尽，也将得到一个int类型结果
int val = 3;
int intR = 23 / val; //得到的结果是 7
