new Date() // 当前日期和时间
显示如：Sat Oct 13 2018 12:16:38 GMT+0800 (中国标准时间)

var d = Date();

d.getTime() 返回从 1970 年 1 月 1 日至今的毫秒数。

d.getFullYear(); // 获取年份
下一个年份： d.getFullYear() + 1

d.getMonth(); //获取月份索引, 0~11
注：月份用索引数字表示，1月份从0开始，10月份返回数字9，所以正确的月份是 d.getMonth() + 1

d.getDate(); //获取日期，返回当前日期数字,1~31

d.getDay(); //获取星期索引，注：周日返回0，周一至周六返回 1~6

d.getHours(); //获取当前的小时 0~23

d.getMinutes(); //获取当前的分钟数，0~59

d.getSeconds(); //获取当前的秒数，0~59
