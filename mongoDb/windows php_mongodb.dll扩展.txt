在 windows上 安装php_mongo.dll扩展很简单：
进入 http://pecl.php.net/package/mongo
下载符合自己的php版本和电脑位数的 扩展，如果不知道应该下载哪一个就先打印一下 echo phpinfo()
查看一下自己的php版本信息：
Architecture	x86 （虽然你的电脑是64位的，但安装的php仍可能是x86的）
Loaded Configuration File	D:\phpstudy\php\php-5.6.27-nts\php.ini （你的php版本的配置文件地址）

把下载好的 php_mongo.dll文件放入你的扩展目录里面，如：D:\phpstudy\php\php-5.6.27-nts\ext\
然后在 php.ini 文件引入扩展：
[mongoDb]
extension=php_mongo.dll

重启服务器，查看 phpinfo()

上面是mongo扩展， 要安装mongoDb扩展的话，进入 http://pecl.php.net/package/mongodb 下载扩展文件，方法和上面一样的
注：最新是mongodb扩展， mongo扩展现已废弃，建议用新的mongodb扩展


另外有一个php扩展mongodb实现了和mongo操作一样的方法，github地址：https://github.com/mongodb/mongo-php-library
文档地址：https://docs.mongodb.com/php-library/current/
