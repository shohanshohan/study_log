php变量规则：
$（美元符号）后面跟变量名，名称以字母或下划线开头，可含字母、数字、下划线，中文，区分大小写。
记住：不能以数字或特殊字符开头！并且变量名中不应有特殊字符！否则报错
可以用中文命名变量名：$变量='中文变量名'; echo $变量;
$this 是一个特殊的变量，它不能被赋值

变量默认总是传值赋值,当一个变量的值赋予另外一个变量时，改变其中一个变量的值，将不会影响到另外一个变量.
如：
$a = 1; 
$b = $a;
$b += 1;
echo $a; // 1, $a的值不变
echo $b; // 2

除了传值赋值，另外一种方式是 引用赋值：& 加变量名
如：
$a = 1;
$b = &$a;
$b += 1;
echo $a; // 2, $a的值也一起改变了
echo $b; // 2
这意味着新的变量简单的引用（换言之，“成为其别名” 或者 “指向”）了原始变量。改动新的变量将影响到原始变量，反之亦然。
又如：
$a = 1;
$b = &$a;
$a += 1;
echo $a; // 2
echo $b; // 2, $b的值也一起改变了

函数引用：
如：
function test(&$a)
{
  $a += 10;
}
$b = 1;
test($b);
echo $b; // 11, 这是因为$b在函数 test() 里做了引用传值

以下方式引用传值不允许：
$bar = &test();    // 非法
$bar = &(24 * 7);  // 非法; 引用没有名字的表达式
只有有名字的变量才可以引用赋值


全局变量，局部变量，静态变量，超全局变量，可变变量

超全局变量：（是PHP系统中自带的变量，在一个脚本的全部作用域中都可用。 你不需要特别说明，就可以在函数及类中使用。共9个）
$GLOBALS   // PHP的一个超级全局变量组
$_GET      // $_GET 应用于收集表单数据，在HTML form标签的指定该属性："method="get"。
$_POST     //  $_POST 应用于收集表单数据，在HTML form标签的指定该属性："method="post"
$_REQUEST  // $_REQUEST 用于收集HTML表单提交的数据
$_COOKIE   // $_COOKIE 变量用于取回 cookie 的值
$_SESSION  // 存储和取回 session 值，把用户信息存储到 PHP session 中之前，首先必须启动会话 session_start()
$_FIELS    // 通过使用全局数组 $_FILES，你可以从客户计算机向远程服务器上传文件
$_ENV      // 获取环境变量值
$_SERVER   // 一个包含了诸如头信息(header)、路径(path)、以及脚本位置(script locations)等等信息的数组

变量范围：（全局变量 / 局部变量）
局部变量限制在局部函数范围内，也就是说函数内定义的变量只在函数内部使用，同样的，函数外定义的变量也不能在函数内部使用，把函数理解为独立的功能块，
当然函数的参数可以使用全局变量。全局变量在函数中使用时必须声明为 global
如：$a = 'a'; $b='b';  function test(){ global $a,$b; echo $a . $b;} 或用$GLOBALS['a'] 也能在函数内访问
函数中嵌入其它函数，各自定义的变量也不能互用，也是各自独立的，如：
function test(){ $a='a'; }  function test2(){ test(); $b=$a;} $a在test2()为未定义,不能这样用

静态变量：（静态变量仅在局部函数域中存在，但当程序执行离开此作用域时，其值并不丢失。定义静态变量 static $var）
静态变量定义不能用函数或表达式，应该设置具体值，如：static $int = $a+$b; static $b = time();这是不允许的。
纯数字数学运算可以 static $int = 1+2;是允许的。但这种情况不行，$a = 3; static $int = 1+$a;这会报错
function test(){ $a = 1; return ++$a;} echo test(); // 输出 2     echo test(); //输出3，第二次调用时静态变量的值会沿用原来的值

可变变量：（一个可变变量获取了一个普通变量的值作为这个可变变量的变量名）
使用了两个美元符号（$）以后，就可以作为一个可变变量的变量了，其中原先被定义的变量值将作为变量名，如：
$a = 'hello';
$$a = ' world';
echo $a.$hello; // 输出 hello world  , echo $a.$$a; 结果也一样
注意超全局变量不能被用作可变变量，$this也一样
